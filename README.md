# Sero_android
Sero는 세로가 긴 영상들을 공유하는 SNS입니다. 
세로가 긴 영상은 모바일이나 태블릿 같은 디스플레이를 가진 디바이스에서 fullscreen으로 영상을 시청하기에 적합합니다.

Sero는
* 뉴스피드
* 검색
* 알림(활동내역)
* 개인공간

과 같은 기본적인 SNS 기능들이 포함되어 있으며

영상 SNS인만큼
* clip 영상 촬영
* 영상 편집

과 같은 기능들도 포함되어 있습니다.
